# My Diary Website

A simple personal diary web app built with **HTML**, **CSS**, and **JavaScript**.  
It lets you create, edit, delete, and search your daily notes — all saved locally in your browser using `localStorage`.

---

## Features
-  Create new diary entries  
-  Save and edit existing entries  
-  Delete unwanted entries  
-  Search through your notes  
-  Auto-save every 10 seconds  
-  Export and Import your entries as `.json` files  
-  Download individual notes as text files  
-  Responsive layout for mobile and desktop  
-  Works completely offline — no server needed  

---

## How It Works
- The app stores your diary entries in the browser’s **localStorage**.  
- You can export your notes as a backup (`.json` file) and import them later.  
- All data stays **on your device**, nothing is uploaded online.

---

## How to Run
1. Download or clone this repository.  
2. Open the file `index.html` in your browser.  
3. Start writing your diary!
